# 搞钱了么 - 个人财务管理与财富增长应用

## 📱 应用介绍

**搞钱了么** 是一款专为个人用户打造的财务管理与财富增长应用，帮助用户实现财务目标，培养良好的理财习惯，开启财富自由之路。 

**最后更新**: 2026年3月1日

### 🌟 核心功能

#### 💰 财富入库（记账模块）
- **智能记账**: 快速记录收入和支出
- **分类管理**: 支持多种收支类别（工资、理财、兼职等）
- **历史查询**: 按月份查看历史记录
- **便捷操作**: 左滑删除记录，操作流畅

#### 🎯 目标模块
- **年度目标**: 设定个人年度搞钱目标
- **实时追踪**: 显示目标完成进度和已到账金额
- **日历功能**: 集成阳历和农历日历
- **每日金句**: 提供365条搞钱金句，激励财富增长
- **宜忌提醒**: 每日财富宜忌提示

#### 🎮 搞钱模块
- **金币奖励**: 完成任务赚取金币
- **月度统计**: 区分上月可提现金币和本月金币
- **便捷提现**: 支持支付宝提现
- **语音通知**: 金币到账语音提醒
- **倒计时功能**: 防止过度点击，保护账户安全

#### 🎰 福利模块
- **幸运抽奖**: 参与幸运奖券抽奖
- **奖金池**: 累积奖金池，中奖几率高
- **实时开奖**: 自动开奖，公平透明
- **多级奖励**: 支持特等奖、一等奖、二等奖、三等奖

#### 🔐 登录系统
- **便捷登录**: 手机号+验证码登录
- **自动注册**: 首次登录自动创建账号
- **持久登录**: 登录状态本地保存
- **安全退出**: 退出登录二次确认

## 🛠️ 技术架构

## 🏗️ 构建和部署

### 使用 GitHub Actions 自动构建

项目已配置 GitHub Actions 工作流，可以自动构建 Android APK。

**触发构建的方式：**
1. 推送代码到 `main` 或 `master` 分支
2. 创建 Pull Request
3. 在 GitHub Actions 页面手动触发

**下载 APK：**
- 进入仓库的 "Actions" 标签
- 选择 "Build Android APK" 工作流
- 下载构建的 APK 文件

详细说明请查看 [BUILD_APK.md](BUILD_APK.md)

### APK 签名配置

如果需要上架应用商店（小米、华为、应用宝等），需要配置 APK 签名。

**配置步骤：**
1. 生成签名密钥库（Keystore）
2. 将密钥库转换为 Base64
3. 在 GitHub Secrets 中配置签名信息
4. 触发构建，自动生成已签名的 APK

详细配置指南请查看 [SIGNING_GUIDE.md](SIGNING_GUIDE.md)

### 本地运行

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

### 本地构建 APK

```bash
# 构建前端
npm run build

# 同步 Capacitor
npm run cap:sync

# 构建 Debug APK
cd android
chmod +x gradlew
./gradlew assembleDebug

# 构建 Release APK（需要配置签名）
cd android
./gradlew assembleRelease
```
